# 苍翼混沌效应 流派构建器

游戏《苍翼：混沌效应》(BlazBlue: Entropy Effect) 的流派构建器，帮助玩家筛选双重词条和构建流派配置。

## 功能特性

- **双重词条筛选**：根据属性/流派/触发位筛选符合条件的双重策略
- **流派构建器**：为5个技能位配置流派，自动计算可激活的双重策略
- **主题切换**：支持深色/浅色模式切换
- **响应式设计**：适配不同屏幕尺寸

## 快速开始

### 环境要求

- Node.js 18+
- pnpm（推荐）或 npm

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

### 预览生产构建

```bash
pnpm preview
```

## 核心概念

### 技能位（5个固定位置）

- 普攻：普通攻击
- 技能：技能攻击
- 冲刺：冲刺技能
- 传承：传承技能
- 召唤：召唤技能

### 属性（7种元素）

火、冰、电、毒、暗、光、刃

### 双重策略

当两个特定流派在正确的技能位组合时，会激活额外的双重策略效果。共73种双重策略。
